1 - Execute the ToClipboard installer. 
    It'll install to C:\ProgramFiles (x86)\ToClipboard. 
    There should be a shortcut put on the desktop as well.
2 - Run the application. 
3 - Right click the application on the taskbar, and pin it there.
4 - Fill in a title and password
5 - link it to a game (copy from an explorer window the full path to the file). 
6 - Click the checkbox to launch the game when clicked.
7 - Save and Exit
8 - Right click the app on the taskbar
9 - Click the title you entered. The app will copy the password to the clipbard and launch the game.
